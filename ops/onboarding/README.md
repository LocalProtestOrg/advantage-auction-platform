# Seller Onboarding Systems

Planning directory for the seller onboarding experience — from first application
to first live auction. Covers psychology, AI-assisted flows, and friction reduction.

The engineering-built onboarding flow lives in the platform (`/seller-create.html`,
seller application routes). This directory plans the *experience layer* on top of that
technical foundation.

---

## Onboarding Phases

```
Phase 1 — Awareness
  Seller encounters Advantage (campaign, SEO, partner referral)
  Goal: recognize Advantage as the right solution for their situation

Phase 2 — Intent
  Seller visits a scenario page or landing page
  Goal: understand what working with Advantage looks like

Phase 3 — Application
  Seller submits the application form
  Goal: low friction, high completion rate

Phase 4 — Assessment
  Ops reviews the application and responds
  Goal: fast, personal, qualified response

Phase 5 — Activation
  Seller is approved, auction is scheduled, photography is booked
  Goal: seller feels confident and supported

Phase 6 — Live Auction
  Auction runs — seller can watch bidding in real time
  Goal: seller trusts the process, sees proof of the platform working

Phase 7 — Payout
  Seller receives funds after pickup window closes
  Goal: full satisfaction, referral opportunity
```
