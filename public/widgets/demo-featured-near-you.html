Featured Auctions Near You - Widget Demo

Production-ready embed for BD and partner pages. Self-contained, zero dependencies, XSS-safe.

Live demo - geo-enabled with seller CTA

Requests the visitor's location (if data-use-geolocation="true"). Fetches featured auctions near them. If permission is denied or no geo-filtered results exist, falls back gracefully to the national featured feed. A seller CTA card appears at the end of the grid.

Embed snippet - basic (no geolocation)

Returns the national featured feed sorted by marketplace priority. No browser permission dialog. Simplest embed - one script tag.

<div id="aap-featured-near-you" data-api-base="https://auctions.advantage.bid" data-limit="6"> </div> <script src="https://auctions.advantage.bid/widgets/featured-near-you.js"></script>

Embed snippet - geo-enabled with radius

Asks the visitor for their location and shows auctions within 200 km. Falls back to the national feed if permission is denied or no nearby results exist.

<div id="aap-featured-near-you" data-api-base="https://auctions.advantage.bid" data-limit="6" data-use-geolocation="true" data-radius-km="200"> </div> <script src="https://auctions.advantage.bid/widgets/featured-near-you.js"></script>

Embed snippet - dark theme with seller CTA

Dark theme switches via a CSS class on the grid - no re-injection of styles. The seller CTA card appears at the end of the results grid.

<div id="aap-featured-near-you" data-api-base="https://auctions.advantage.bid" data-limit="6" data-use-geolocation="true" data-radius-km="300" data-theme="dark" data-seller-cta-url="https://auctions.advantage.bid/seller-create.html" data-seller-cta-headline="Consigning an Estate?" data-seller-cta-label="Start Consigning"> </div> <script src="https://auctions.advantage.bid/widgets/featured-near-you.js"></script>

Embed snippet - with shared utils (multiple widgets on one page)

Load shared/utils.js before both widget scripts. Utilities are shared in memory - only one copy of esc(), fmtDate() etc. regardless of how many widgets are on the page.

<!-- Utilities - load once, before any widget --> <script src="https://auctions.advantage.bid/widgets/shared/utils.js"></script> <!-- Widget A: Featured Near You --> <div id="aap-featured-near-you" data-api-base="..." data-use-geolocation="true"></div> <script src="https://auctions.advantage.bid/widgets/featured-near-you.js"></script> <!-- Widget B: Featured Auctions (original) --> <div id="aap-featured-auctions" data-api-base="..."></div> <script src="https://auctions.advantage.bid/widgets/featured-auctions.js"></script>

Analytics event integration

The widget fires CustomEvents that bubble from the container element. Listen for them on the container or any ancestor (including window). No polling required. Events carry structured detail objects for downstream analytics and tag manager integration.

var container = document.getElementById('aap-featured-near-you'); // Fires when the widget finishes rendering container.addEventListener('aap:widget:loaded', function (e) { console.log('Loaded', e.detail); // { widgetId: 'aap-featured-near-you', resultCount: 6, source: 'featured' } // source: 'featured' | 'near' | 'national' }); // Fires when the widget switches from the primary to a fallback data source container.addEventListener('aap:widget:fallback', function (e) { console.log('Fallback triggered', e.detail); // { reason: 'geo-denied' | 'geo-unavailable' | 'geo-timeout' | 'no-results' } }); // Fires when a visitor clicks an auction card container.addEventListener('aap:auction:click', function (e) { console.log('Card clicked', e.detail); // { auctionId, title, distanceKm, source } // Route the visitor to your auction detail page using auctionId. }); // Fires when a visitor clicks the seller CTA button container.addEventListener('aap:cta:click', function (e) { console.log('CTA clicked', e.detail); // { widgetId: 'aap-featured-near-you' } });

Configuration reference

Attribute Default Description
data-api-base (same origin) API host, e.g. https://auctions.advantage.bid
data-limit 6 Number of auction cards (1-12)
data-use-geolocation false Set to true to request browser location
data-radius-km 200 Search radius in km when geolocation is active (1-800)
data-geo-timeout-ms 5000 Geolocation request timeout in ms (1000-15000)
data-theme light light or dark
data-seller-cta-url (none) Seller CTA card link. Omit to hide the CTA card entirely.
data-seller-cta-headline Consigning an Estate? CTA card headline text
data-seller-cta-label Learn More CTA button label

Fetch strategy

ConditionPrimary endpointFallback endpointsource
Geo granted + featured results exist nearby /api/public/featured-auctions?lat=…&lng=… - featured
Geo granted + no featured results nearby /api/public/featured-auctions?lat=…&lng=… (empty) /api/public/auctions/near?lat=…&lng=… near
Geo denied / unavailable / timed out /api/public/featured-auctions - national
Geolocation not requested (data-use-geolocation omitted) /api/public/featured-auctions - national
