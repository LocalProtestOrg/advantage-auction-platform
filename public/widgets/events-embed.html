Local Events: embed on Brilliant Directories

Paste one of these into a BD city page. Events are managed on the Advantage platform; BD only displays them. Markets available: houstonnyc_tristate

1) JavaScript widget (recommended)

Renders event cards inline, isolated from the page’s styles. Change data-market per city page.

<div data-advantage-events data-market="houston" data-limit="12"></div>
<script async src="https://bid.advantage.bid/widgets/events.js"></script>

2) Iframe fallback

For pages where a <script> can’t be added. Auto‑posts its height to the parent.

<iframe src="https://bid.advantage.bid/widgets/events.html?market=nyc_tristate"
  style="width:100%;border:0;min-height:900px" loading="lazy" title="Local events"></iframe>

3) “Create Event” button (deep‑link into Advantage)

Sends organizers to Advantage to create an event (sign‑in/sign‑up if needed; their organization is created automatically). ?market= pre‑selects the city.

<a href="https://bid.advantage.bid/org/events/new?market=houston">Create Event</a>
