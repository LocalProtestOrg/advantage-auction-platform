Featured Auctions Widget — Embed Demo

Copy the snippet below to embed Advantage Auction featured listings on any BD page.

Basic embed (no geolocation)

Renders the top featured auctions sorted by marketplace priority. No browser permission required.

<div id="aap-featured-auctions" data-api-base="https://auctions.advantage.bid" data-limit="6"> </div> <script src="https://auctions.advantage.bid/widgets/featured-auctions.js"></script>
Preview — Basic (no location)

"Near Me" embed (browser geolocation)

Asks for the user's location and filters to auctions within the specified radius. Falls back to national feed if permission is denied.

<div id="aap-featured-auctions" data-api-base="https://auctions.advantage.bid" data-limit="6" data-use-geolocation="true" data-radius-km="200"> </div> <script src="https://auctions.advantage.bid/widgets/featured-auctions.js"></script>

Dark theme

<div id="aap-featured-auctions" data-api-base="https://auctions.advantage.bid" data-limit="4" data-theme="dark"> </div> <script src="https://auctions.advantage.bid/widgets/featured-auctions.js"></script>

Configuration reference

Attribute Default Description
data-api-base (same origin) API host, e.g. https://auctions.advantage.bid
data-limit 6 Number of cards to display (1–12)
data-use-geolocation false Set to true to request browser location
data-radius-km 200 Search radius in km when geolocation is enabled
data-theme light light or dark
