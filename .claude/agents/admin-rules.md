# Admin Override Rules

Advantage admin is the superuser role.

## Always Preserve
- Ability to edit any auction
- Ability to edit any lot
- Ability to add or remove lots
- Ability to replace images
- Ability to change featured lots
- Ability to change buyer premium
- Ability to change seller commission
- Ability to change starting bid defaults and increment schedules
- Ability to enable or disable shipping, reserve, and other seller capabilities
- Ability to add miscellaneous charges with descriptions
- Ability to process refunds
- Ability to recover underbidder information after non-payment
- Ability to save updates multiple times
- Ability to publish and unpublish auctions
- Ability to edit notification behavior, marketing tiers, and campaign settings over time

## Never Break
- Seller lock after final submission must not lock admin
- Public publishing must remain admin-controlled
- Important financial settings must remain editable
- Full address must remain hidden until verified payment
